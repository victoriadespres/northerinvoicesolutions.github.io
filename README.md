# Northern Invoice Solutions Website

Professional static website for Northern Invoice Solutions.

## Upload instructions

Upload the following to your GitHub repository root:

- index.html
- style.css
- script.js
- assets folder

Commit directly to the main branch.

## Contact form

The contact form currently uses a placeholder Formspree URL:

https://formspree.io/f/your-form-id

Replace it with your actual Formspree endpoint when ready.
